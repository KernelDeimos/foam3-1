# exchange-rate
Receives an amount in a currency as input, returns the exchange rate for another currency as output

# build & install
sh build.sh

# using
add
```
<dependency>
    <groupId>net.nanopay</groupId>
    <artifactId>exchange-rate</artifactId>
    <version>1.0</version>
</dependency>
```
to your pom.xml dependencies
